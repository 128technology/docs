Mosh Plugin

The Mosh plugin provides the ability to install the MObile SHell MOSH on SSR nodes. Mosh is provided under GNU GPLv3.

important

This plugin has been deprecated with the release of SSR 6.3.0.

note

The instructions for installing and managing the plugin can be found here.

Configuration​

The configuration to manage this plugin is located in the node configuration

*admin@t133-dut1.conductor (node[name=TNode])# show
name          TNode
asset-id      t133-dut2.openstacklocal
role          combo

mosh
    enabled  true
exit
